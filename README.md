# StudyBuddy Grade Dashboard

A powerful, local-first web application for tracking Dutch school grades (Somtoday), calculating graduation goals, and managing practice tests.

## Features
- **Somtoday Integration**: Import `.stgrades` backup files.
- **Graduation Dashboard**: Track SE averages from 10th grade onwards.
- **CE Goal Calculator**: Calculate what you need on final exams to reach your target grades.
- **Practice Tracker**: Log and visualize practice test performance.
- **Accomplishments**: Record your academic milestones.
- **Fully Local**: No data is uploaded to servers; everything stays in your browser's local storage.
- **Mobile Optimized**: Premium dark-themed UI that works perfectly on phones.

## How to use
1. Open `index.html` in any modern browser.
2. Upload your `.stgrades` file.
3. Switch to the **Graduation** tab to set your goals.
4. Use the **Settings** tab to export/backup your data.

## Technologies
- HTML5 / Vanilla CSS
- Vanilla JavaScript
- Chart.js for visualizations
